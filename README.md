# Responsive-Landing-page
Drone Website



## Description

This repository contains the source code for a modern drone website landing page. It features a fully responsive design with sections such as Home, About, Products, Gallery, Blog, Contact, and a subscription form.

## Features

- Fully responsive design optimized for all screen sizes
- Smooth-scrolling navigation menu
- Dedicated sections for Home, About, Product, Gallery, Blog, and Subscription
- Scroll reveal animations for an engaging user experience
